# Knowledge base index

> The four tables below are generated by **Dataview**. As long as each page's frontmatter is filled in correctly (`summary` / `venue_short` / `year` / `type` / `source_count`), the tables update live when you add or delete pages — no manual maintenance needed.
>
> Scale statistics (computed live by Dataview):

```dataview
TABLE WITHOUT ID
  file.folder AS "Section",
  length(rows) AS "Pages"
FROM "wiki/concepts" OR "wiki/entities" OR "wiki/sources" OR "wiki/synthesis"
GROUP BY file.folder
SORT file.folder ASC
```

## Concepts

```dataview
TABLE WITHOUT ID
  file.link AS "Page",
  summary AS "Summary",
  source_count AS "Sources"
FROM "wiki/concepts"
SORT source_count DESC, file.name ASC
```

## Sources (newest first)

```dataview
TABLE WITHOUT ID
  file.link AS "Page",
  summary AS "Title",
  venue_short AS "Venue",
  year AS "Year"
FROM "wiki/sources"
SORT year DESC, file.name ASC
```

## Entities

```dataview
TABLE WITHOUT ID
  file.link AS "Page",
  type AS "Type",
  summary AS "Summary"
FROM "wiki/entities"
SORT type ASC, file.name ASC
```

## Synthesis

```dataview
TABLE WITHOUT ID
  file.link AS "Page",
  summary AS "Topic",
  source_count AS "Cited sources"
FROM "wiki/synthesis"
SORT created DESC
```
